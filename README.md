# Tamizhi Programming Language

Tamizhi is a Tamil-based simple programming language.

## Structure

- lexer.py : Tokenizes Tamil source code.
- parser.py : Parses tokens into AST.
- compiler.py : Compiles AST into bytecode.
- tamizhi_vm.c : C virtual machine to run bytecode.
- tamizhi_cli.py : CLI to compile and run Tamizhi code.
- stdlib/ : Standard library files.
- tests/ : Test programs.

## Usage

1. Write Tamizhi code in `.tml` file.
2. Run CLI tool to compile and execute.
3. The VM executes the bytecode.
